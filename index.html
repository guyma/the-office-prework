⚠️חשוב מאוד:
מנוי בתשלום חובה. אי אפשר לעבוד עם המנוי החינמי.
אני רוצה שבסדנה ממש תעבדו ותבנו על העסק האמיתי שלכם.
בשביל זה הכול צריך להיות מותקן, ואתם צריכים להכין את כל החומרים של העסק בתיקייה.

תתחילו כמה ימים לפני הסדנה, לא ערב לפני. בלי ההתקנות לא תוכלו לעבוד בסדנה וחבל, זה כל הכיף!

חשבון Claude בתשלום

כ-5 דקות

אני מלמד בסדנה לעבוד עם Claude Code, וזה דורש מנוי בתשלום: Pro (20$ לחודש) ומעלה. למי שבא לעבוד ברצינות, ההמלצה שלי: Max. תזהרו, זה ממכר 😉

  • נכנסים אל claude.ai ולוחצים Sign up. נרשמים עם המייל שלכם או עם חשבון Google. רואים: מסך השיחה של Claude, אתם בפנים.
  • פותחים את תפריט החשבון (לוחצים על השם או על סמל הפרופיל), בוחרים Settings, ומחפשים שם Billing או Plan. זה העמוד שמראה על איזה מסלול אתם.
  • כתוב שם Free? לוחצים Upgrade, בוחרים Pro או Max, וממלאים פרטי תשלום. בסיום, באותו עמוד יהיה כתוב Pro או Max.

סיימתם כשבעמוד ה-Billing של claude.ai כתוב Pro או Max.

מתקינים את Claude Code

כ-10 דקות
  • נכנסים אל claude.com/download ולוחצים על כפתור ההורדה ל-Windows. הקובץ יורד לתיקיית Downloads.
  • לוחצים פעמיים על הקובץ שירד ומחכים שההתקנה תסתיים. בסיום Claude נפתח לבד, או שמוצאים אותו בתפריט Start.
  • לוחצים Sign in. נפתח דפדפן עם עמוד התחברות: מתחברים עם החשבון מהמשימה הקודמת ומאשרים. חוזרים לחלון של Claude (אם הוא לא קפץ לבד, לוחצים עליו בשורת המשימות למטה).
  • בחלק העליון של החלון, באמצע, יש שלוש לשוניות: Chat, Cowork, Code. לוחצים על Code. נפתח מסך עבודה שמציע לבחור תיקייה, ובלי שום בקשת שדרוג. זה הבית של העובד שלכם.
Windows מציג חלון אזהרה כחול על הקובץ?

זה מסך הבדיקה של Windows (SmartScreen). לוחצים More info ובודקים את שורת המפרסם: Publisher: Anthropic, PBC. זה מה שכתוב? לוחצים Run anyway. כתוב שם אחר, או שאין שם מפרסם? עוצרים, לא מריצים, וכותבים לי.

סיימתם כשלשונית Code נפתחת ומציעה לבחור תיקייה, בלי בקשת שדרוג.

מתקינים את Obsidian ופותחים Vault

כ-10 דקות
  • נכנסים אל obsidian.md/download ולוחצים על כפתור ההורדה ל-Windows.
  • לוחצים על הקובץ שירד. אם נפתחים מסכי התקנה, לוחצים Next בכולם בלי לשנות כלום, ובמסך האחרון Finish. בסיום האפליקציה נפתחת לבד.
  • במסך הפתיחה לוחצים Create new vault (כספת היא פשוט תיקייה שבה יחיו הקבצים שלכם).
  • נותנים שם באנגלית ובלי רווחים, למשל: my-office.
⚠️את ה-vault שומרים בתיקיית הבית שלכם במחשב (C:\Users\your-name).
לא בתוך תיקייה שמסונכרנת לענן כמו OneDrive, Google Drive או iCloud.
בהרבה מחשבים תיקיית Documents מסונכרנת ל-OneDrive בלי שיודעים, לכן תיקיית הבית עצמה הכי בטוחה.
  • בוחרים את המיקום, לוחצים Create, ונפתח vault ריק.
    זוכרים איפה שמתם אותו: נשתמש בו עוד מעט בבדיקה, וגם בסדנה.

סיימתם כש-Obsidian פתוחה עם ה-vault החדש שלכם (my-office).

מתקינים את Git

כ-5 דקות
⚠️התקנת חובה: בלעדיה לשונית Code לא עובדת על התיקיות שלכם.
  • נכנסים אל git-scm.com/downloads/win ולוחצים על קישור ההורדה הראשון (Click here to download).
  • מריצים את הקובץ שירד. אם מופיעה אזהרת מפרסם: החותם הרשמי של Git for Windows הוא Johannes Schindelin. זה השם? ממשיכים. שם אחר? עוצרים ושואלים אותי.
  • יגיעו הרבה מסכי אפשרויות בזה אחר זה. לוחצים Next בכולם בלי לשנות כלום, גם במסכים שנראים מסובכים (בחירת עורך, שם ה-branch). בסוף לוחצים Finish.
  • סוגרים את Claude לגמרי ופותחים מחדש, כדי שיזהה את Git. אם נשאר סמל של Claude באזור השעון, קליק ימני עליו ואז Quit לפני הפתיחה מחדש.
ההתקנה הזו נחשבת זמנית עד שמשימה 7 עוברת: אם הבדיקה שם רצה עד הסוף, Git תקין אצלכם.

סיימתם כשההתקנה הגיעה ל-Finish ו-Claude נסגר ונפתח. האימות הסופי: משימה 7.

פותחים חשבון GitHub

כ-5 דקות

בסדנה נשתמש בחשבון הזה לגיבוי הפרטי של המשרד שלכם. החשבון חינמי.

  • נכנסים אל github.com/signup וממלאים מייל, סיסמה ושם משתמש.
  • GitHub שולח קוד אימות למייל, מזינים אותו (לא הגיע? בודקים בספאם).
  • אם מוצע לכם להגדיר אימות דו-שלבי (2FA), עוקבים אחרי ההנחיות על המסך ושומרים את קודי הגיבוי במקום בטוח.
  • ולבדיקה שחשובה לסדנה: מתנתקים (Sign out מתפריט הפרופיל) ומתחברים שוב, כולל שלב האימות אם הגדרתם. נכנסתם בלי להיתקע? מצוין.

סיימתם כשהתנתקתם והתחברתם מחדש בהצלחה.

הבדיקה: הרגע הראשון עם העובד

כ-10 דקות

המשימה הכי חשובה בהכנה. אם היא עוברת, כל המערכת שלכם עובדת: מהמנוי ועד הקובץ ב-Obsidian.

  • פותחים את Claude ונכנסים ללשונית Code.
  • מוודאים שסביבת העבודה היא Local ושמצב האישורים הוא Manual. זו ברירת המחדל, אז ברוב המקרים אין מה לשנות.
  • מחפשים את כפתור Select folder, לוחצים, ובוחרים את תיקיית ה-vault שיצרתם במשימה 3 (my-office).
  • בתיבת הכתיבה מקלידים בדיוק: צור קובץ בשם hello.md ובתוכו ברכת שלום קצרה בעברית. ולוחצים Enter.
  • קלוד חושב רגע ואז מציג הצעת שינוי: קובץ חדש בשם hello.md והטקסט שהוא רוצה לכתוב, עם כפתורי Accept ו-Reject.
    זה לא תקוע, ככה עובדים איתו: הוא מציע, אתם מאשרים.
    קוראים את ההצעה ולוחצים Accept עליה. לא מפעילים אישור גורף לכל דבר, גם לא בהמשך.
  • ועכשיו הקסם: פותחים את Obsidian. הקובץ hello.md מופיע ב-vault שלכם, עם הברכה בפנים. זה בדיוק החיבור שנעבוד איתו בסדנה, והוא כבר עובד אצלכם.
הודעה שמזכירה Git?

חוזרים רגע למשימה 4: אתחול מלא למחשב, ואם ההודעה חוזרת, מריצים שוב את מתקין Git עד Finish, סוגרים ופותחים את Claude ומנסים שוב. עדיין תקוע? כתבו לי.

קלוד עצר ולא ממשיך?

רוב הסיכויים שהוא פשוט מחכה לאישור שלכם. מחפשים על המסך הצעה עם כפתור Accept ולוחצים עליו.

קלוד סיים אבל אין קובץ ב-Obsidian?

כנראה נבחרה תיקייה אחרת. בודקים איזו תיקייה מוצגת כפעילה בלשונית Code: אם זה לא ה-vault שלכם, לוחצים Select folder, בוחרים אותו, ומבקשים שוב.

סיימתם, חד וחלק: hello.md מופיע ב-Obsidian ויש בו את הברכה שביקשתם.

שאלון ההקמה

בקצב שלכם

ממלאים שאלון על העסק, ובסוף מורידים קובץ מוכן. ממנו העובד יכיר אתכם.

לשאלון ההקמה
  • אפשר לעצור באמצע ולחזור להמשיך, באותו מחשב ואותו דפדפן.
  • בסוף לוחצים על כפתור ההורדה. הקובץ my-business-he.md יורד לתיקיית Downloads (לא מוצאים? Ctrl ו-J ביחד פותח את רשימת ההורדות של הדפדפן).
  • מעבירים אותו למקום שקל למצוא, שולחן העבודה מצוין. לא משנים את שם הקובץ, ומביאים אותו לסדנה.

סיימתם כשהקובץ my-business-he.md שמור אצלכם במקום שתמצאו.

תיקיית הדוגמאות

כ-15 דקות

פותחים תיקייה אחת במחשב ואוספים אליה דברים שאתם כתבתם.
מהדוגמאות האלה העובד לומד את הקול שלכם.

⚠️כלל אחד שמלווה את כל האיסוף: שם או פרט רגיש של לקוח שלא צריך לחיות בקובץ, מחליפים ל-[שם] כבר עכשיו, בזמן האיסוף.
  • פוסטים שפרסמתם: פייסבוק, אינסטגרם, לינקדאין.
  • הצעת מחיר ששלחתם.
  • שיחות וואטסאפ עם לקוחות. גם כאן, שם רגיש מחליפים ל-[שם].
  • מיילים ללקוחות, ניוזלטר, סיכום פגישה.
  • וגם קבצים של העסק: מחירון, חוזה, ההודעה שאתם שולחים שוב ושוב.

לפחות 3, אין תקרה. דוגמה טובה היא טקסט שכתבתם בעצמכם, כמו שהוא, בלי לערוך וליפות. עדיף קובץ טקסט על צילום מסך, ועדיף ייצוא צ'אט על תמונות של השיחה. לא צריך לסדר כלום, בסדנה נלמד את העובד מהכול.

ייצוא צ'אט מוואטסאפ? ככה:

באנדרואיד: פותחים את השיחה, תפריט שלוש הנקודות, More ואז Export chat, בוחרים Without media, ושולחים לעצמכם למייל.
באייפון: פותחים את השיחה, לוחצים על השם של איש הקשר או הקבוצה למעלה, גוללים ל-Export Chat, ובוחרים Without Media.
מורידים את הקובץ מהמייל אל תיקיית הדוגמאות.

סיימתם כשפותחים את התיקייה ורואים בה לפחות 3 דוגמאות שלכם.

לא לבד

נתקעתם באחת המשימות? כתבו לי בוואטסאפ מה ניסיתם ומה רואים על המסך (צילום מסך עוזר מאוד). לכל סדנה יש גם קבוצת וואטסאפ, את הקישור תקבלו ממני עם ההרשמה. והזום נפתח חצי שעה לפני הסדנה: מי שרוצה עזרה בהתקנות, בואו מוקדם ונסדר הכול ביחד. ודבר אחרון: מחשב של עבודה שחוסם התקנות? לא נלחמים בו, מדברים עם ה-IT שלכם או איתי.
